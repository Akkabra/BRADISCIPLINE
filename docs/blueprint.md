# **App Name**: Bra Discipline

## Core Features:

- Personalized Dashboard: Display a personalized greeting, progress tracker, consecutive days completed, and a daily motivational quote to start the day with focus.
- Daily Routine Guide: Step-by-step checklist with tasks such as cold shower, focus routine, exercise, project time, and nightly reflection, each with completion buttons and inspirational text.
- Nocturnal Journal: Record daily thoughts and personal progress. Record also the emotional state (expressed as a emoji). Allow entries to be saved by date and reviewed weekly.  The LLM will use a tool to flag trends (positive/negative) based on keywords or repeated topics that emerge over time, helping the user derive insight from this journal.
- Progress Statistics: Visually track the user's progress using graphs and timelines to display daily compliance, growth over time, and unlocked achievements.
- Parenthood Focus: Schedule dedicated time with children, list 'connection tasks' (reading together, park outings), provide advice on positive masculinity, and record emotional experiences with their child.
- Cloud Data Sync: Register/Login with email/google and save data to Firebase for synchronization between devices.
- Notifications: Configure notifications to maintain consistent focus.

## Style Guidelines:

- Primary color: Matte gold or polished copper (#C9A227) for a strong accent.
- Background color: Carbon black (#0D0D0D) to evoke a masculine, metallic, and minimalist aesthetic.
- Accent color: Deep dark green (#357A38) for success and controlled blood red (#A83232) for alerts.
- Headline font: 'Bebas Neue' sans-serif for firm, vertical, masculine titles. Note: currently only Google Fonts are supported.
- Body font: 'Inter' sans-serif for a clean, modern general text. Note: currently only Google Fonts are supported.
- Use geometric, minimalist 'outline' style icons (Lucide or Phosphor Icons).
- Implement smooth hover animations, such as a golden underline or soft illumination, and transitions with Framer Motion (fade-in, slide, subtle glow).
